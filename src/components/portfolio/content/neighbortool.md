## Overview

NeighborTool is a community-driven platform designed to make tool access easy and sustainable. By connecting neighbors who have tools with those who need them, the platform promotes resource sharing, reduces unnecessary consumption, and strengthens local communities.

Users can list their available tools, search for items nearby, and arrange secure borrowing transactions, all within a trusted neighborhood network.

## Tech Stack

- **Frontend**: Next.js 14, React, TailwindCSS, Shadcn UI
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Maps**: Mapbox GL JS for location-based search
- **Payment**: Stripe Connect for security deposits and rentals
- **Image Storage**: Supabase Storage with optimization
- **Deployment**: Vercel

## Key Features

- **Location-Based Search**: Find tools available within your specific neighborhood radius
- **Secure Verification**: Identity verification system for trusted borrowing
- **Real-time Availability**: Calendar integration to check tool availability
- **In-app Messaging**: Secure communication between lenders and borrowers
- **Smart Notifications**: Reminders for returns and pickup coordination
- **Community Ratings**: Review system to build trust within the network
- **Tool Management**: Easy inventory tracking for lenders
